Crystal Collector!

Instructions:

You will be given a random number below.

Underneath your random number, are four crystals, all having a random number value at the start of each round.
Click each crystal to view its value and try to match the total value of the crystals clicked to the number generated at the top.
At the end of each round, a new random number will be generated and the crystals will reset to new number values as well.

Good Luck!
